# Data Mesh Contracts Platform — Thesis Landing Page

This is the public-facing landing page for my thesis project: **Building a Real-World Data Mesh Contracts Platform in 4 Months**. The project involves designing and implementing a production-grade platform where domain teams can register governed data products, publish events through Apache Kafka, and expose datasets via a contracts API, CLI, and MCP tooling — making enterprise data discoverable and AI-ready. This landing page documents the active build, showcasing the platform's architecture, the eight domain producers already wired (MediaWiki, Jira, GitHub, Kimai, Paperless, Vtiger, Dovecot, and a generic OIDC producer), a live 4-month roadmap tracker, and links to the main repository.
